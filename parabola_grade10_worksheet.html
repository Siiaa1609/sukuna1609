Key Formulas & Definitions
Vertical (up/down): (x−h)² = 4p(y−k)
Horizontal (left/right): (y−k)² = 4p(x−h)
Vertex: (h, k)  ·  Focus: p units from vertex
Directrix: perpendicular to axis, p units opposite focus
Opens up: p > 0  ·  Opens down: p < 0
Opens right: p > 0  ·  Opens left: p < 0
Latus Rectum length: |4p|
Axis of symmetry (vert.): x = h

Answer Key & Solutions

Grade 10 — Parabola Practice Worksheet  ·  For teacher / self-evaluation use

Problem 1  x² = 12y
4p = 12 → p = 3 (positive → opens up)
Vertex: (0, 0)
Focus: (0, 3)
Directrix: y = −3
Axis of symmetry: x = 0 (y-axis)
Latus rectum length: |4p| = 12
Problem 2  y² = −20x
4p = −20 → p = −5 (opens left)
Vertex: (0, 0)
Focus: (−5, 0)
Directrix: x = 5
Latus rectum length: |4p| = 20
Problem 3  (x−3)²=8(y+2)
4p = 8 → p = 2 (opens up)
Vertex: (3, −2)
Focus: (3, −2+2) = (3, 0)
Directrix: y = −2−2 = y = −4
Axis of symmetry: x = 3
Latus rectum length: 8
Problem 4  (y+4)²=−12(x−1)
4p = −12 → p = −3 (opens left)
Vertex: (1, −4)
Focus: (1−3, −4) = (−2, −4)
Directrix: x = 1+3 = x = 4
Axis of symmetry: y = −4
Problem 5  Vertex origin, Focus(0,−4)
Focus below vertex → opens down · p = −4
4p = −16
x² = −16y
Problem 6  Vertex(2,5), Directrix y=2
Directrix below vertex → opens up
p = distance from vertex to directrix = 5−2 = 3
4p = 12
(x − 2)² = 12(y − 5)
Problem 7  Focus(−1,3), Directrix x=5
Focus left of directrix → opens left (horizontal parabola)
Vertex = midpoint of focus and foot of directrix on axis
h = (−1+5)/2 = 2, k = 3 → Vertex: (2, 3)
p = 2−(−1) = −3 (focus left of vertex) → 4p = −12
(y − 3)² = −12(x − 2)
Problem 8  x²−6x−8y+1=0
(x²−6x) = 8y − 1
(x−3)²−9 = 8y−1 → (x−3)² = 8y+8
(x − 3)² = 8(y + 1)
Vertex: (3, −1) · p = 2
Focus: (3, 1)
Directrix: y = −3
Problem 9  y²+10y−4x+9=0
(y²+10y) = 4x−9
(y+5)²−25 = 4x−9 → (y+5)² = 4x+16
(y + 5)² = 4(x + 4)
Vertex: (−4, −5) · p = 1 · opens right
Focus: (−3, −5)
Directrix: x = −5
Axis of symmetry: y = −5
Problem 10
(a) TRUE — By definition, vertex is equidistant from focus and directrix, so it is the midpoint.
(b) FALSE — A parabola has exactly one axis of symmetry.
(c) TRUE — The latus rectum is a chord through the focus perpendicular to the axis.
Problem 11  Satellite Dish
Dish is 8 m wide → edge point is (4, 2). Vertex at origin.
Parabola opens up: x² = 4py. Point (4, 2) lies on it.
16 = 4p(2) → 4p = 8 → p = 2
x² = 8y
Receiver (focus): (0, 2) — 2 m above the vertex
Problem 12  (x+1)²=−8(y−3)
4p = −8 → p = −2 (opens downward)
Vertex: (−1, 3)
Focus: (−1, 1)
Directrix: y = 5
Axis of symmetry: x = −1
Latus rectum: length 8, endpoints (−5, 1) and (3, 1)
Sketch: downward-opening curve, vertex at (−1,3), dotted horizontal directrix at y=5, focus dot at (−1,1), vertical axis line x=−1, horizontal latus rectum segment at y=1.
